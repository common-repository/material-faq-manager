== Description ==

This plugin is design and developed to display faq page content with latest material design as well as popup answer.

= More Features =

*   Extra custom post type.
*   Shortcode enable. 
*   Latest material design.
*   Answer popout.
*   Easily display anywhere of your webpage.
*   Fully responsive design

This plugin is suitable for, Who wants to give a latest material design to FAQ and help page.

== Installation ==

**Through Admin Area**

1. Login to your wordpress admin area and go to plugins menu and click to add new plugin sub menu.
2. Type **Material FAQ Manager** in the plugin page search box and hit enter.
3. Find **Material FAQ Manager** plugin
4. Click the install now button and activate the plugin.
5. Go to FAQ menu and add new FAQ.
6. Shortcode to display **[material_fm_list]**.

**Through Ftp**

1. Download the plugin to your computer.
2. Unzip the plugin zip file.
3. Upload the material-faq-manager folder in to the plugin directory via ftp or filezilla.
4. Login to your admin dashboard and click on the plugin menu.
5. Activate the plugin.
6. Shortcode to display **[material_fm_list]**.
